Capítulo 0 - Introdução#

Bem vinda!#

Este compilado de documentos de programação em python surgiu do nivelamento em machine learning do grupo Ada, do Instituto Santos Dumont. Nosso objetivo é fazer com que pessoas interessadas no tema rapidamente aprendam o básico de programação e possam contribuir com projetos do instituto.

Qual o nosso objetivo?#

A programação é fundamental para o processamento de dados de qualquer natureza, em especial os dados eletrofisiológicos, comportamentais e logísticos do Instituto Santos Dumont.

O objetivo deste caderno é aumentar a acessibilidade dos alunos do programa de mestrado do IIN-ELS a técnicas de análise através de código python.

A programação vai muito além disso#

Talvez você se pergunte: eu preciso mesmo aprender algo que só vou usar no mestrado? Você sobreviveu bem até aqui sem programação, mas este é um campo imenso, e saber programar vai te dar acesso a técnicas e experiências que podem ser úteis além do mestrado.

Por exemplo: em vagas de doutorado que atuam em eletrofisiologia e análise de dados, muitas vezes é pedido que a candidata saiba montar seus próprios códigos de análise.

Indo além, programação se extende a automatizar processos digitais (ex. gerar documentos de forma automática), gerenciar sistemas de venda, organizar arquivos, programar máquinas e dispositivos, fazer sites e aplicativos.

A gente do grupo Ada sempre gosta de lembrar onde tudo isso começou: com a matemática e escritora inglesa Ada Lovelace (1815 - 1852). Precursora da computação, trabalhando como assistente de Charles Babbage (1791 - 1871) no Reino Unido, ele responsável por desenvolver uma máquina analítica capaz de realizar cálculos complexos por meio da interação entre engrenagens e manivelas, e ela repsonsável por programá-la. Na verdade não existia esse conceito de programação, Ada Lovelace foi a primeira programadora da história.

Estrutura do caderno#

Vamos começar pelo básico de lógica de programação, um tema que permeia o fundamento de qualquer linguagem de computador. A partir dessa visão, vamos entender o que são algoritmos e só então trataremos de uma linguagem específica: o python.

Assim como outras linguagens de programação, o conteúdo básico para você aprender a programar é:

  • Tipos de variáveis

  • Estruturas

  • Condicionais

  • Loops

  • Funções

  • Classes

  • Bibliotecas

Depois disso, você pode se aprofundar nesses temas, ou seguir em frente com conteúdos mais avançados do seu interesse.

Bons estudos!
